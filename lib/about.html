About the lib folder

The lib folder contains code (usually in the form of modules) that can be shared among many styles. You can use the shared code in your styles in one of the following ways -

  1. By copying the file into your own style and loading it using -
    (load (resource "the-file.scm"))
  2. By directly loading it from the web using -
    (load (fetch-uri "http://muvee-style-authoring.googlecode.com/svn/trunk/lib/the-file.scm"))
