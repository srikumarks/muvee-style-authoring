6.1 muSE

The behaviour of muvee styles in Reveal is specified in a scripting language we call “muSE” - which is short for “muvee Symbolic Expressions”. muSE is a dialect and subset of a family of programming languages collectively called “Scheme”. Each muvee style has a "data.scm" file containing muSE code that specifies its behaviour.

  • See muvee-symbolic-expressions on googlecode for details on the open sourced core language.

  • See A Gentle Introduction to muSE if you aren’t familiar with muSE or Scheme, or for a quick recap if you haven’t used them in a while.

  • Post questions about core muSE to muSE’s own forum

  • Post questions about use of muSE for style authoring to the style authoring forum.
