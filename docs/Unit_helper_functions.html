7.10 Unit helper functions

pi : 3.141592654
The world famous constant.

(deg->rad degrees) → number
  degrees : number
Converts degrees into radian units.

(rad->deg radians) → number
  radians : number
Inverse of deg->rad.

(beat->sec beats tempo-bpm) → number
  beats : number
  tempo-bpm : number
Converts a time interval expressed in beats into seconds, at the given tempo.
