6.3 Title and Credits

(title-section ...) → void
(credits-section ...) → void
These sections specify the default settings for the title and credits configurable settings and decide the particular look that the style author wants to use for them that aligns with the style.

Below is a template title-section which illustrates all the possible settings that can be specified. The same structure goes for credits-section as well.

(title-section
    ; Background description, 4 possible types shown below,
    ; but use only one!
    ; 1. a picture, typically jpg
    (background (image "personal1.jpg"))
 
    ; 2. solid background color, in RGB format, [default]
    (background (color 0 0 0))
 
    ; 3. video clip:See note below on usage of video files
    (background (video "intro_30.avi"))
 
    ; 4. arbitrary effect.
    (background (color 0xFFFFFF)
                (fx ...arbitrary effect to use as background ...))
 
    ; Foreground object.
    (foreground (fx ...global effect...))
 
    ; Optional background audio file.
    ; The keyword arguments are optional.
    (audio-clip "personal.mp3" gaindb: 0.0 fadein: 0.0 fadeout: 0.0)
 
    ; Text description
    (text
        ; Text's color, in RGB format [default]
        (color 255 255 255)
 
        ; Font - using Win32's LOGFONT structure
        (font "-32,0,0,0,500,0,0,0,0,3,2,1,2,Batik Regular")
 
        ; Layout rectangle, in normalised coords
        ; [default is TV-safe region]
        (layout (0.1 0.1) (0.9 0.9))
 
        ; Text animation, You can use zero or more of these.
        (fade-in duration: d
                 initial: 0.0
                 final: 1.0)
        (fade-out duration: d
                  initial: 1.0
                  final: 0.0)
        (zoom-in)
        (zoom-out)
        (typewriter)
        (color-reverse)
 
        ; Basic shadow effects, choose one [default is no shadow]
        (soft-shadow dx: 1.0 dy: 1.0 size: 3.0)
        (hard-shadow dx: 1.0 dy: 1.0 size: 3.0)
        (inner-shadow dx: 1.0 dy: 1.0 size: 3.0)
 
        ; Text alignment - horizontal=left|center|right
        ; and vertical=top|center|bottom
        (align 'center 'center)
 
        ; Besides the above, you can explicitly specify any parameter
        ; exposed by [[TextOut]] using <param> tags.
        (custom-parameters
            (param "TextFade" 0.0
                (at (effect-time 0.0) 0.0)
                (linear (effect-time 2.0) 0.0)
                (linear (effect-time 3.0) 1.0)))))
